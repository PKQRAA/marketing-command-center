## Features (8 Modules)

| Module | Description |
|--------|-------------|
| **Full Scan** | Scan all at once - CTAs, Emails, Prices, Social, SEO, Links |
| **CTAs** | Find all CTAs, headlines, buttons with one click |
| **Emails** | Extract emails from text and mailto links |
| **Prices** | Extract prices with Min/Max/Avg statistics |
| **Social** | Find all social media profiles |
| **SEO** | Full SEO analysis with score and keywords |
| **Links** | Analyze all internal/external links |
| **Meta Tags** | View all meta tags (OG, Twitter, etc.) |

## NEW: PDF Report

Now generates **REAL PDF files** - not text files!

Click **Download PDF Report** to get a professionally formatted PDF with:
- SEO Score
- All CTAs
- All Emails
- All Prices
- All Social Links

## Installation

1. Go to `chrome://extensions/`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select `marketing-command-center` folder

## How to Use

1. Open any website (e.g., amazon.com)
2. Click the extension icon
3. Click **Full Scan** or any individual tool
4. Click **Copy** on any result to copy
5. Click **Download PDF Report** for PDF export
